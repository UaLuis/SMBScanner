# SMBScanner
Scanner for SMB

How to use:

Use a Kali Linux.

1. ```git clone https://github.com/UaLuis/SMBScanner.git && cd SMBScanner```
2. ```sudo apt update && sudp apt install enum4linux -y```
3. ```chmod +x smbscanner.sh```
4. Make a file target.txt woth IP range or IP
5. ```sudo ./smbscanner.sh```
